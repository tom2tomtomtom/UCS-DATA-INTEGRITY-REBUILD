# Synthetic Law Fixtures

Use this folder for small hand-built fixtures that prove individual immutable
laws without relying on live source data.

Examples:

- source-only row visibility.
- unsupported metric represented as unsupported.
- zero as a source-supported value.
- raw row that is not additive.
- archive state visible as a warning.

Synthetic fixtures must still name the law they protect and must not contain
real client exports.
